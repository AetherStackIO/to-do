## To-Do List Web App

# 🎯 Goal of the project
    You will build a website where you can:
    
    add tasks (example: “study”, “clean the room”)
    see all your tasks
    delete tasks when they are done
    
    It’s basically a digital checklist.
----------------------------------------------------------------------------------
  
# 🧩 STEP-BY-STEP: What you need to do
    1. 🏗️ Build the basic structure (HTML idea)
    
    Think of your page as having 3 main parts:
    
    Title (example: “My To-Do List”)
    Input box (where you type a task)
    Button (to add the task)
    List area (where all tasks will appear)
    
    👉 Goal: “Create a place where users can type and see tasks”
    
    2. 🎨 Simple design (CSS idea)
    
    Keep it basic and clean:
    
    Center the app on the screen
    Put everything inside a card-like box
    Add spacing so it looks neat
    Make the button visible with color
    Style the list items so they look organized
    
    👉 Goal: “Make it clean and easy to use”
    
    3. ⚙️ App logic (JavaScript idea)
    
    This is the “brain” of your app:
    
    A. When you click “Add”
    Get the text from the input box
    Show it in the list
    B. If the input is empty
    Show a warning: “Please enter a task”
    C. If there is text
    Add it to the task list
    4. 🗑️ Delete feature (upgrade)
    
    Each task should have:
    
    a delete button (X)
    
    When clicked:
    
    the task will be removed from the list
    5. 🧪 Testing (very important)
    
    Try:
    
    adding 5 tasks
    deleting tasks one by one
    trying to add an empty task (should show error)
    refreshing the page (check if tasks disappear)
    🚀 FINAL OUTPUT OF THE PROJECT
----------------------------------------------------------------------------------
  
When finished, your app should be able to:

✔ add tasks
✔ display tasks
✔ delete tasks
✔ have a simple design
